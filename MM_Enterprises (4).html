Cloud Sync — Multi-Device Access

Not Connected

Connect a free Supabase database to sync your data across all devices — phone, tablet, PC. Once connected, every bill, stock entry and payment is instantly available everywhere.

Open Supabase.com
📋 One-time Setup SQL — Run this in Supabase SQL Editor:
CREATE TABLE IF NOT EXISTS mm_data (
  id TEXT PRIMARY KEY,
  value JSONB,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE mm_data ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON mm_data
  FOR ALL USING (true) WITH CHECK (true);
After creating the table, paste your Project URL and anon key in the Connect screen above.
